# Jasleen Kaur — Portfolio

Personal portfolio site built from resume content. Showcases ML/AI projects, skills, education, and contact information.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy

The `dist/` folder can be deployed to [Vercel](https://vercel.com), [Netlify](https://netlify.com), or GitHub Pages.

For GitHub Pages, set `base` in `vite.config.ts` to your repo name if needed.

## Customize

Edit `src/data/resume.ts` to update projects, skills, and contact details.
